# Transitland USA working lists

Generated locally from the public [Transitland Atlas](https://github.com/transitland/transitland-atlas) for research.

| File | What |
|------|------|
| `transitland-usa-operators.csv` | USA NTD operators (gitignored — regenerate locally) |
| `transitland-usa-open-vp.csv` | Open vehicle-position subset (gitignored) |

**Do not commit these CSVs.** Atlas rows sometimes embed third-party `apiKey`/`token` query params. Regenerate and scrub before any sharing.

**tier** values: `open_vp` | `auth_vp` | `static_only`

Live inventory remains the Google Sheet; feed URLs must be verified before enable.
